# Mini-Git Study

Study notes and exercises for building [Mini-Git](https://github.com/kwanele1/Mini-Git) — a simplified version control system.

Work through the folders **in order**. Each section maps directly to a phase of the build.

## Learning Path

| # | Topic | Needed For |
|---|-------|-----------|
| 01 | [C# Foundations](./01-csharp-foundations/) | Everything |
| 02 | [Hashing / SHA-256](./02-hashing/) | Phase 1 |
| 03 | [Data Structures](./03-data-structures/) | Phases 1–3 |
| 04 | [Recursion](./04-recursion/) | Phases 2–3 |
| 05 | [Algorithms (BFS/DFS, LCA, Myers)](./05-algorithms/) | Phases 4–5 |
| 06 | [Serialization](./06-serialization/) | Phase 3 |
| 07 | [Clean Architecture](./07-clean-architecture/) | Phase 3+ |
| 08 | [CLI Development](./08-cli-development/) | Phase 1 |
| 09 | [Testing with xUnit](./09-testing/) | Every phase |
| 10 | [HTTP & GitHub API](./10-http-github-api/) | Phase 6 |
| 11 | [Git Internals](./11-git-internals/) | Context for all |

## Folder Convention

Each section has:
- `notes.md` — your notes on the topic
- `exercises/` — scratch code or practice files
