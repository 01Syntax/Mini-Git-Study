# 03 — Data Structures

Understand these structures before building the object model. Each one maps to a Mini-Git concept.

| # | Structure | Mini-Git use |
|---|-----------|-------------|
| 01 | [Hash Map](./01-hash-map/notes.md) | Object store, staging index |
| 02 | [Linked List](./02-linked-list/notes.md) | Linear commit history |
| 03 | [Tree](./03-tree/notes.md) | Directory snapshots (TreeObject) |
| 04 | [DAG](./04-dag/notes.md) | Full commit history with merges |
